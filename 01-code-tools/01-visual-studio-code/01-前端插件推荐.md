 ## CSS

自动补全样式信息, 让你编写样式表的时候, 大大缩减了编写时间
```shell
HTML CSS Support
```

对使用了 css module 的 jsx 标签的类名补全和跳转到定义位置
```
CSS Modules
```

面板着色, 我其实也没试过, 实际作用是什么不知道, 嘿嘿
```shell
Output Colorizer
```

px 转换为 rem 等比例, 自己设置比例即可
```shell
cssrem
```

当我们创建一个.less文件, 写完代码进行保存时, 会发现它会在.less文件同目录生成对应名字的.css文件, 以css样式生成
```shell
Easy LESS
```
